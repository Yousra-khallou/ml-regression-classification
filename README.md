# Machine Learning – Regression & Classification Projects

This repository contains two end-to-end machine learning projects demonstrating
both regression and classification tasks.

## Projects Overview

### 1. Medical Cost Prediction (Regression)
- Objective: Predict medical expenses using regression models
- Techniques:
  - Data preprocessing and feature engineering
  - Model comparison
  - Cross-validation
  - Hyperparameter tuning
- Evaluation metrics: RMSE, MAE, R²

### 2. Weather Prediction (Classification)
- Objective: Predict weather conditions using classification models
- Techniques:
  - Binary classification
  - Model evaluation using precision, recall, and ROC curves
  - Hyperparameter tuning (SVM)
- Evaluation metrics: Accuracy, Precision, Recall, ROC-AUC

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Key Learning Outcomes
- Building complete ML pipelines
- Comparing multiple algorithms
- Selecting appropriate evaluation metrics
- Fine-tuning models for optimal performance
